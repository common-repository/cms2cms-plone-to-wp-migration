Plugin for automated data migration from Plone to WordPress. Convert Plone to WordPress easily and fast.

== Description ==

If you wish to switch from Plone CMS and consider WordPress a perfect solution for your current web project – CMS2CMS Migration Plugin can more than help you to go through the migration path with no programming knowledge, much time and effort.

= HOW THIS PLUGIN HELPS: =
- Ease-of-Use – CMS2CMS is a totally user-oriented tool, that doesn’t require any special skills to master it;
- High Speed – usually a full-scale migration with CMS2CMS is a matter of 15 min to a couple of hours (depending on a website size);
- Free Demo Available – CMS2CMS provides its users with an opportunity to see a website migration in action by importing a limited number of content to a target platform;
- Complete Migration price depends on how many content items you wish to move to WordPress.


= WHAT WILL BE MOVED FROM PLONE TO WORDPRESS? =
* Pages along with the content included there;
* Attachments (images and videos);
* Images included in the content body.


= PRESERVED SEO JUICE AFTER THE MIGRATION =
* Migration of SEO friendly URLs;
* Migration of keywords, metadata/meta description;
* Automated 301 redirects from existing Plone site to the new WordPress website.


= COMPATIBLE WITH: WORDPRESS 4.1.X – 5.0.X. =
- Our support team is available live 24/7.
- For more customized migration type, CMS2CMS offers data migration service packages to order.

== Installation ==
1. Download the plugin zip file.
2. Go to the Admin Panel -> Plugins -> Add New -> Upload Plugin.
3. Choose the downloaded file in zip format and click the “Install Now” button.
4. Go to Admin -> Plugins, activate the “CMS2CMS: Automated Plone To WordPress Migration Plugin”.
5. Register an account or sign in to the existing one.
6. Provide your Existing website URL address.
7. Click ‘Verify Connection’. You’ll be redirected to the CMS2CMS website to proceed with your migration.
